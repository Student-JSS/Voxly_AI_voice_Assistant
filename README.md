# 🎙️ Voxly AI Voice Assistant

Voxly is a modern AI-powered Voice Assistant SaaS platform that enables users to interact with advanced AI models through natural voice conversations. It combines Speech-to-Text, AI processing, and Text-to-Speech to deliver a seamless real-time conversational experience.

## 🚀 Live Demo

🔗 https://voxly-ai-voice-assistant-2.onrender.com/login

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Voxly_AI_Voice_Assistant.git
```



### Install Frontend Dependencies

```bash
cd Client
npm install
```

### Install Backend Dependencies

```bash
cd ../Server
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the server directory:

```env
PORT=8000

MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_jwt_secret

RAZORPAY_KEY_ID=your_key_id
RAZORPAY_KEY_SECRET=your_secret_key

```

---

## ▶️ Run Locally

### Start Backend

```bash
cd Server
npm run server
```

### Start Frontend

```bash
cd Client
npm run dev
```

Application will run on:

```bash
Frontend: http://localhost:5173

Backend: http://localhost:8000
```
